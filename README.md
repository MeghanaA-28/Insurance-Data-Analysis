# 🛡️ Insurance Data Analysis | Data Analytics Project  

## 🚀 Project Overview  
Performed end-to-end data analysis on an insurance dataset to identify key factors influencing medical insurance charges. The project focuses on analyzing customer demographics and lifestyle attributes such as age, BMI, smoking habits, and region.  


## 🎯 Objective  
To uncover patterns in insurance data and generate actionable insights that support risk assessment and pricing strategies.  



## 🛠️ Tools & Technologies  
- Excel → Data cleaning and preprocessing  
- SQL → Data querying and analysis  
- Power BI / Tableau → Data visualization and dashboard creation  



## 📊 Key Analysis Performed  
- Data cleaning and handling missing values  
- Exploratory Data Analysis (EDA)  
- Correlation analysis between variables  
- Impact analysis of age, BMI, smoking, and region on charges  



## 💡 Key Insights  
- Smoking significantly increases insurance charges  
- Age is positively correlated with higher medical costs  
- Higher BMI contributes to increased expenses  
- Regional differences impact pricing patterns
